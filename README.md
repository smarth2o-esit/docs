## ¿Qué contiene este repositorio?

Este repositorio centraliza toda la documentación técnica y de gestión del proyecto SmartH2O: bitácoras semanales, documentos de análisis y diseño, evidencias por semana, informe final y archivos de referencia del proyecto.

## Estructura de una bitácora semanal

Cada archivo en `bitacoras/` sigue el formato oficial ESIT con las secciones:

1. Información general de la semana
2. Actividades realizadas
3. Herramientas y servicios utilizados
4. Problemas detectados y análisis
5. Soluciones aplicadas
6. Estado del proyecto
7. Plan de trabajo para la próxima semana
8–13. Aportes y cumplimiento individual
14–15. Registro de evidencias
